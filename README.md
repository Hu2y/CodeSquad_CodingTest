### Step-1
```
코드 동작
1. 게임 시작버튼으로 게임을 시작합니다.
2. 게임 시작되면 시작버튼은 비활성화로 되고 진행버튼을 클릭하여 게임을 진행합니다.
3. 게임 시작시 첫 타자 입장 메시지가 나오고 SBO는 초기 값으로 세팅합니다.
4. 진행버튼을 클릭할때마다 야구 룰에 따라 SBO가 출력됩니다.
5. 3out이 누적되면 게임이 종료되고 종료메시지와 함께 누적 안타수가 나타납니다.
```

#### Step-2, Step-3
```
*중복되는 코드가 많아 step2,3 데이터를 같이 사용했습니다.

Step-2 코드동작
*화면은 각 스텝에 맞게 나타나게 됩니다.(요구사항 적용합니다.)

1. 야구팀을 먼저 입력합니다.(step=1~2)
   - 첫번째 팀 명과, 선수 9명을 입력받습니다.
   - 그다음 두번째 팀 명과 선수 9명을 입력받습니다.
2. 두 팀 입력이 완료되면, 팀정보를 볼수있는 버튼이 활성화 되고, 입력이 완료되었다는 메시지와 함께 게임시작 버튼이 나타납니다.(step3)
3. 게임시작 버튼을 누르면 시합시작 메시지와 첫회, 1번팀 1번타자가 나타나고 SBO는 초기값으로 세팅합니다.
*진행 버튼이 활성화됩니다. 야구팀 입력버튼이 비활성화됩니다.
4. 진행버튼을 클릭할때마다 SBO가 출력되고, 야구 룰에따라 게임이 진행됩니다. 타격 결과는 확률을 적용합니다. 
5. 6회말까지 경기가 끝나게되면 최종 스코어와 경기 종료 메시지가 나타납니다.

Step-3 코드동작
*Step-2 진행방식과 동일합니다

1. 야구팀 입력이 완료되면 스킵버튼이 활성화 됩니다.
 - 스킵할 라운드 수와 버튼을 누르면 스킵이됩니다(0<x<7)
 - 선수 입력이 되면 바로 선수의 번호와 이름이 전광판에 출력됩니다. 
 - 타율은 0<h<1 사이로 입력되며 소숫점을 길게 입력해도 3자리 까지 자동으로 줄어들게됩니다.
2. 진행버튼으로 동일하게 진행되며, 각 라운드별 점수가 전광판에 기록됩니다. 
3. 해당 선수가 바뀔떄마다 V표시가 따라오고, SBO는 X의 갯수로 나타납니다.
4.총 투구수가 아래 표시되고 투수가 없다는걸 전제로, 피안타수, 피삼진으로 기록했습니다.
5. 경기가 종료되면 step-2 영역에 같은 종료 메시지를 나타내었습니다.
```
